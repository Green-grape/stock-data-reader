# Creon-Datareader-Mongo
    대신증권의 HTS인 CREON의 Plus API를 사용하여 주가 데이터를 받아오는 PyQt 기반의 프로그램이며 nestjs와의 연동을 위해서
    MongoDB를 선택했습니다.

# 실행 환경 세팅
    CREON Plus가 32bit 환경에서 지원되기 때문에 32bit Python이 필요하므로 Anaconda 32-bit 환경에서 구동하는 것을 추천합니다.
